# Amazon Cognito is now available in Asia Pacific (Taipei) and Asia Pacific (New Zealand) Regions

Amazon Cognito is now available in the AWS Asia Pacific (Taipei) and Asia Pacific (New Zealand) Regions. This launch introduces all Amazon Cognito features and tiers, allowing customers to implement secure sign-in and access control for users, AI agents, and microservices in minutes.

For a full list of regions where Amazon Cognito is available, refer to the [AWS Region Table](https://aws.amazon.com/about-aws/global-infrastructure/regional-product-services/).

To learn more about Amazon Cognito, refer to the [Developer Guide](https://docs.aws.amazon.com/cognito/latest/developerguide/what-is-amazon-cognito.html), [Product Detail Page](https://aws.amazon.com/cognito/), and [Pricing Detail Page](https://aws.amazon.com/cognito/pricing/).
